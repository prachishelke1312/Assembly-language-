# X86/64 Assembly Language Programs

This repository contains X86/64 Assembly Language Programs (ALP) implemented using NASM on Linux for Microprocessor Laboratory practicals.

---

## 📌 Programs Included

1. Accept and display five 64-bit hexadecimal numbers
2. Find length of a string
3. Count positive and negative numbers
4. Multiplication using successive addition
5. Find largest number from array
6. Menu driven arithmetic operations
7. HEX to BCD conversion
8. BCD to HEX conversion
9. Study of motherboard components

---

## 🛠 Technologies Used

- X86/64 Assembly Language
- NASM Assembler
- Linux System Calls
- Ubuntu/Linux Environment

---

## 📚 Concepts Covered

- Arrays and Pointer Handling
- Loops and Conditional Statements
- Arithmetic Operations
- String Handling
- Number System Conversion
- ASCII Conversion
- System Calls (`read`, `write`, `exit`)
